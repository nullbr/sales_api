## Sales API

Working with FastAPI to practice creating an API with python.

### Dependencies

- python ~> 3.10
- pip ~> 21.3

### Installation

$ python3 -m pip install --user --upgrade pip
$ python3 -m pip install --user virtualenv
$ python3 -m venv venv
$ source venv/bin/activate
$ python3 -m pip install "fastapi[all]"
